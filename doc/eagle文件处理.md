### metadata.json 示例结构：

```json
{
  "id": "KBHG6KA0Y5S9W",           // 唯一标识符
  "name": "Rosto.jpg",             // 文件名
  "ext": "jpg",                    // 扩展名
  "width": 1920,                   // 宽度
  "height": 1080,                  // 高度
  "size": 2048576,                 // 文件大小(字节)
  "color": "#FF5733",              // 主色调
  "palettes": [                    // 调色板
    "#FF5733",
    "#33FF57",
    "#3357FF"
  ],
  "rating": 4,                     // 星级评分
  "tags": ["人物", "肖像"],        // 标签
  "folders": ["portraits"],        // 文件夹
  "annotation": "客户肖像照",      // 注释
  "importedAt": 1712345678901,    // 导入时间
  "modifiedAt": 1712345678901,     // 修改时间
  "website": "https://...",        // 来源网址
  "noThumbnail": false,            // 是否有缩略图
  "noPreview": false               // 是否有预览
}
```

## 完整流程图

```
用户拖入图片
      │
      ▼
┌──────────────────────────────────────────────────────────────────┐
│  Step 1: 文件识别     → 检测格式、验证完整性                     │
├──────────────────────────────────────────────────────────────────┤
│  Step 2: 数据读取     → 加载原始文件内容                          │
├──────────────────────────────────────────────────────────────────┤
│  Step 3: 缩略图生成   → 2000px / 400px / 100px 三级缓存          │
├──────────────────────────────────────────────────────────────────┤
│  Step 4: 元数据提取   → 尺寸、颜色、EXIF、调色板                 │
├──────────────────────────────────────────────────────────────────┤
│  Step 5: 生成唯一ID   → UUID 格式唯一标识符                      │
├──────────────────────────────────────────────────────────────────┤
│  Step 6: 创建存储     → 原始文件 + metadata.json + 缩略图        │
├──────────────────────────────────────────────────────────────────┤
│  Step 7: 数据库索引   → SQLite 写入、标签关联                   │
├──────────────────────────────────────────────────────────────────┤
│  Step 8: AI 处理      → 以图找图索引、语义理解（可选）           │
├──────────────────────────────────────────────────────────────────┤
│  Step 9: 完成导入     → 返回 Item 实例、可立即浏览               │
└──────────────────────────────────────────────────────────────────┘
```

## 存储空间占用

| 组件 | 占用比例 | 说明 |
|------|----------|------|
| 原始文件 | 原始大小 | 保持不变 |
| 缩略图 | ~3-5% | 多级缓存 |
| 元数据 JSON | <1% | 轻量数据 |
| 数据库索引 | <1% | 结构化数据 |

> **总计**：额外约 **5%** 的存储空间用于缓存和索引

## 总结

Eagle 导入资产时的核心操作：

1. **读取** → 加载原始文件
2. **生成** → 多级缩略图
3. **分析** → 提取元数据
4. **存储** → 原始文件 + metadata.json
5. **索引** → SQLite 数据库
6. **智能** → AI 特征提取（可选）
