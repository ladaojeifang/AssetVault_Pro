# 常见问题（FAQ）

---

## 安装与启动

### Q：支持哪些操作系统？

**A**：暂时只支持windows（具体安装包见发布页）。


### Q：启动后异常或端口冲突？

**A**：

1. 完全退出后重启
2. 检查 41596 端口是否被占用（Web API 默认端口，可在设置中修改）
3. 若数据库损坏，查看日志中 `library.sqlite.corrupt-*` 相关说明

---

## 资料库

### Q：archive 和 catalog 有什么区别？

**A**：

- **archive**：文件复制/硬链接到资料库内，推荐大多数用户
- **catalog**：仅记录外部路径，省空间但依赖原文件位置

### Q：catalog 能升级为 archive 吗？

**A**：可以。设置 → 资料库 →「转为完整库」，会将引用文件复制进 `items/`。需足够磁盘空间，且不可逆。

### Q：embedded 模式有什么限制？

**A**：文件必须保持在原路径；设置中**没有**「从其他资料库导入」按钮；不能像 archive 那样随意搬迁 `items/`。

### Q：资料库数据存在哪？

**A**：资料库根目录下的 `library.sqlite`、`manifest.json`、`items/` 等，全部为本地文件，不上传服务器。

## 导入

### Q：支持哪些格式？

**A**：150+ 种扩展名，涵盖图片、视频、音频、字体、设计稿、3D、文档、代码等。完整列表见 [asset-types-and-import.md](../asset-types-and-import.md)。

### Q：导入时提示重复怎么办？

**A**：

- **桌面端**：弹出对话框，可选使用已有、再导入一份或取消；可勾选「应用于本批其余文件」
- **设置中无法配置默认策略**
- **Web API**：默认 `use_existing`（使用已有），可传 `duplicatePolicy: "import_copy"`

### Q：能从菜单里 URL 导入或下载 B 站/YouTube 视频吗？

**A**：桌面端工具栏**没有**这些菜单。请使用浏览器扩展或 Web API（`importFromURL`、`pageVideoImport` 等）。

### Q：URL 导入失败？

**A**：检查链接是否可直接下载、是否需要 Referer（API 请求体可指定）、网络是否正常。作品页须用 `pageVideoImport` 而非普通 URL 导入。

### Q：文件夹导入可以中断吗？

**A**：关闭应用可中断；已入库的文件会保留。

## 浏览与搜索

### Q：搜索哪些字段？

**A**：文件名、原始文件名、标签、备注、来源链接。多词 AND 匹配。

### Q：缩略图不显示？

**A**：

- 格式不支持或文件损坏
- 大 SVG（>5 MB）跳过栅格缩略图
- 首次导入后异步生成中，稍候或按 `F5` / `Ctrl+R` 刷新
- 尝试 `Ctrl+Alt+R` 刷新缩略图，或在设置 → 资料库中重生成对应类型缩略图

### Q：Space 和双击有什么区别？

**A**：`Space` 打开详情面板（须单选）。双击进入全页预览（若支持）或打开资源管理器。

## 预览

### Q：3D 预览卡顿？

**A**：关闭其他占 GPU 程序；超大模型较慢属正常。`.blend`/`.c4d` 等仅有嵌入缩略图，无交互 3D 视口。

### Q：字体预览中文异常？

**A**：设置 → 资料库 → 字体样例文字，加入中文如 `你好世界 永 AaBbCc 123`。

### Q：代码文件能全页阅读吗？

**A**：当前仅 **Markdown** 有全页阅读/编辑。其他代码格式只有文本预览缩略图。

## AI 画布

### Q：生成需要联网吗？

**A**：v0.5.0-alpha 使用**本地 Mock**（`runNodeGenMock`），不需要联网，也不扣积分。

### Q：积分从哪来？

**A**：当前版本无积分系统；未来接入 AI 网关后见 [AI_GATEWAY_SPEC.md](../AI_GATEWAY_SPEC.md)。

### Q：结果不满意能重跑吗？

**A**：可以调整节点参数后再次点击运行。

## Web API

### Q：安全吗？

**A**：默认仅监听本机且无需 Token。开启远程访问后务必设置强 Token，仅在可信网络使用。

### Q：其他设备能访问吗？

**A**：需在设置中开启「允许远程访问」，此时绑定 `0.0.0.0` 并要求 Bearer Token。

## 性能与故障

### Q：应用变慢？

**A**：

1. 超大资料库占用更多内存
2. 调低设置 → 高级 → 最大缓存，或缩小缩略图尺寸
3. 不需要时关闭 Web API
4. 重启应用

### Q：缩略图占磁盘多？

**A**：降低缩略图质量/尺寸（设置 → 通用）。磁盘 WebP 缓存随资料库保存在本地。

### Q：catalog 资产打不开？

**A**：设置 → 资料库 →「检查引用」；缺失文件用详情面板「重新链接」或「本地化」。

### Q：资料库自检报告什么？

**A**：仅对比 `assets` 表行数与 `items/` 目录数，不检查引用文件或缩略图完整性。

## 其他

### Q：数据会丢吗？

**A**：不会自动云同步；请自行备份资料库文件夹。

### Q：支持多语言吗？

**A**：支持 **简体中文** 与 **English**，在设置 → 外观 → 语言切换，立即生效。

### Q：如何反馈 Bug？

**A**：通过 GitHub Issues，附上系统版本、应用版本、复现步骤与截图。
