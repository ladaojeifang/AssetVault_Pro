# 文件夹、标签与类型

文件夹、标签和**类型**用于从不同维度组织与筛选资产，可组合使用。

---

## 对比

| 特性 | 文件夹 | 标签 | 类型（系统） | 类型（用户分类） | 扩展名 |
|------|--------|------|--------------|------------------|--------|
| 结构 | 树形，最多 **5 层** | 扁平 | 固定 9 类 | 扁平 | — |
| 关联 | 多对多 | 多对多 | 每资产 **一个** `type_id` | 每资产 **一个** `type_id` | 按 `assets.extension` |
| 创建入口 | 侧栏右键 | 详情面板输入框 | —（系统内置） | 侧栏「新建分类」 | — |
| 筛选 | 进入文件夹 | 侧栏多选 OR | 侧栏 / 列表列头 | 侧栏 / 列表列头 | 列表列头下拉或输入 |
| 可重命名 | 是 | 是（API） | 否 | 是 | 否 |
| 适合 | 项目 / 阶段 | 属性 / 状态 | 按有效类型浏览 | 自定义归档 | 按文件后缀精确筛选 |

侧栏 **「类型」** 区块将系统格式与用户分类列在同一列表；数据层均为 `CategoryItem`（系统项 id 形如 `__sys:image`）。入库默认 `type_id = __sys:{检测格式}`，用户可改为自定义分类；**检测格式**（`file_type`）仍驱动缩略图与预览管线。

---

## 类型（系统 + 用户）

### 创建用户分类

侧栏类型区块底部 **「新建分类」**，或详情面板类型区域快速创建。

### 设置资产类型

| 方式 | 说明 |
|------|------|
| 详情面板 | 类型下拉框（系统 + 用户）；可「恢复为检测格式」 |
| 右键菜单 | 「设置类型」子菜单 |
| 导入 API | 不支持内联 `typeId`；合并资料库时按名称迁移 `type_id` |

每个资产**仅有一种**有效类型；重新设置会覆盖原类型。

### 编辑与删除用户分类

用户分类在侧栏 **右键** → 编辑 / 删除。删除分类后，原归属该分类的资产会恢复为 `__sys:{file_type}`。

### 筛选

侧栏点击切换；可多选，多个类型之间为 **OR**（匹配 `type_id`）。与标签、文件夹、搜索等其它条件组合时仍为 AND。

---

## 文件夹

### 结构示例

```
我的项目
├── 概念
├── 模型
└── 渲染
```

- 最多 **5 层**嵌套
- 同一资产可属于**多个**文件夹
- 文件夹可设置封面（浏览该文件夹时，右键资产 → 设为封面）

### 创建与重命名

侧栏文件夹树 **右键** → 新建子文件夹 / 重命名 / 删除 / 设置颜色与图标。

### 关联资产

| 方式 | 说明 |
|------|------|
| 详情面板 | 文件夹下拉框 |
| 右键菜单 | 「添加到文件夹」 |
| 拖放 | 向侧栏文件夹拖放时按住 **Alt**；主区域子文件夹卡片可直接拖放 |

---

## 标签

- **扁平**结构，无层级
- 同一资产可有**多个**标签
- 侧栏多选标签筛选为 **OR**
- 在详情面板输入名称创建或关联；侧栏标签用于筛选

---

## 组合筛选示例

「文件夹 A」+ 标签「参考」+ 类型「图片」→ 在 A 内、带参考标签、且 `type_id` 为 `__sys:image` 的资产。

若某 PNG 在详情里被标为自定义分类「概念稿」，则**不会**出现在仅筛选「图片」系统类型时（除非同时选中「概念稿」）。

---

## Web API

单资产导入后调用 `POST /category/assign` 设置 `typeId`；`POST /category/remove` 重置为检测格式。整库合并自动迁移类型。详见 [Web API](web-api.md)。
