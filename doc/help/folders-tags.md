# 文件夹与标签系统

文件夹和标签用于从不同维度组织资产，可组合使用。

---

## 对比

| 特性 | 文件夹 | 标签 |
|------|--------|------|
| 结构 | 树形，最多 **5 层**（level 0–4） | 扁平 |
| 关联 | 多对多 | 多对多 |
| 创建入口 | 侧栏右键 | 详情面板输入框 |
| 筛选 | 进入文件夹 | 侧栏点击标签（多选 OR） |
| 适合 | 项目 / 类别 | 属性 / 状态 / 风格 |

---

## 文件夹

### 结构示例

```
📁 我的项目
  ├── 📁 角色设计
  │   ├── 📁 主角
  │   └── 📁 NPC
  └── 📁 场景
```

第 5 层（level 4）文件夹不能再创建子文件夹。

### 创建与编辑

侧栏文件夹树 **右键菜单**：

| 操作 | 说明 |
|------|------|
| 新建子文件夹 | 在当前文件夹下创建（父级达 level 4 时禁用） |
| 重命名 | 修改名称 |
| 编辑图标 | Emoji 或导入图片作为图标 |
| 编辑颜色 | 自定义文件夹颜色 |
| 删除 | 删除文件夹结构；**不删除**资产，仅清除关联 |

当前版本**没有**「移动到」其他父文件夹的菜单项。

### 将资产加入文件夹

| 方式 | 说明 |
|------|------|
| 详情面板 | 文件夹下拉框添加 |
| 右键菜单 | 「添加到文件夹」子菜单 |
| 拖放到侧栏文件夹 | 需按住 **Alt** |
| 拖放到主区域子文件夹卡片 | 直接拖放 |
| 导入时 | 若正在浏览某文件夹，新导入资产自动归入 |

### 文件夹封面

浏览某文件夹时，对图片资产或任意有缩略图的资产使用右键菜单 →「设为文件夹封面」。

文件夹图标也可通过右键 →「编辑图标」单独设置（Emoji / 图片）。

---

## 标签

### 特点

- 标签名在资料库内唯一
- 创建时分配颜色；侧栏显示使用计数
- 多资产可共享同一标签

### 创建与关联

**唯一 UI 入口**：选中资产 → 详情面板标签输入框 → 输入名称 → 回车。

- 名称已存在：关联到该标签
- 名称不存在：创建新标签并关联

### 筛选

侧栏标签列表点击切换筛选；可多选，结果为 **OR**（包含任一所选标签即显示）。

### 移除关联

详情面板点击标签上的 ×；**不会**删除标签定义本身。

### 编辑 / 删除标签

桌面端**没有**标签管理界面。标签的更新、删除可通过 Web API（`PATCH /api/v1/tag/update`、`DELETE /api/v1/tag/delete`）完成，但主界面未提供对应入口。

## 组合示例

```
📁 文件夹（按阶段）
  ├── 概念阶段
  ├── 制作阶段
  └── 交付

🏷️ 标签（按状态）
  ├── 高优先级
  ├── 待修改
  └── 已完成
```

在「制作阶段」文件夹内，再筛选「待修改」标签，即可交叉定位素材。
