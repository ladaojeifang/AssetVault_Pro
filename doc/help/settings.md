# 设置系统

AssetVault Pro 的设置以模态窗口呈现，包含 **通用**、**资料库**、**外观**、**快捷键**、**高级** 五个标签页。

打开方式：标题栏齿轮图标，或按 `Ctrl+,`。

> 除 **语言** 与 **主题** 会立即生效外，其余选项需点击窗口底部的「保存」才会写入配置。

---

## 通用

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 默认导入路径 | 部分导入对话框的默认浏览目录（并非所有导入流程都会使用） | 空 |
| 自动监控文件夹 | 导入后监控源文件夹的增删改，并同步到资料库 | **开启** |
| 缩略图质量 | WebP 压缩质量，范围 10–100，步进 5 | 80 |
| 缩略图尺寸 | 缩略图最大边长：128 / 256 / 384 / 512 像素 | 256 |
| 格式图标覆盖 | 为无缩略图的扩展名指定 Emoji 或占位图 | 无 |

### 自动监控文件夹

开启后，每次从本地路径导入文件，应用会监控该文件所在文件夹的变化；新增、修改或删除文件时，资料库会尝试同步。

### 格式图标覆盖

在通用标签页底部，可为特定扩展名自定义网格中显示的图标（Emoji 或图片占位），并恢复 3D 等格式的默认列表。

---

## 资料库

资料库标签页集中管理当前资料库与维护工具，而不仅是路径设置。

### 当前资料库

- 显示模式（archive / catalog / embedded）、显示名称与根目录路径
- 统计本地副本数、引用数，以及 catalog 模式下缺失源文件数量
- 操作按钮：
  - **在资源管理器中打开**
  - **打开其他资料库**（文件夹选择器）
  - **从其他资料库导入**（仅 archive 或 catalog 当前库可用；合并文件夹、标签、有效类型 `type_id`，SHA-256 去重）
  - **转为完整库**、**检查引用**（仅 catalog 模式）

### 新建资料库

- **新建完整库（archive）**：在空文件夹初始化
- **新建索引库（catalog）**：在空文件夹初始化
- **新建内嵌库（embedded）**：选择已有项目文件夹，原地扫描入库

### 最近使用的资料库

列出最近打开的资料库路径，可切换、打开文件夹或从列表移除。

### 内容指纹扫描

扫描资料库内资产并更新 SHA-256 内容哈希，用于导入去重。点击「开始扫描」执行。

### 字体样例文字

配置字体缩略图与预览使用的样例文字；修改后需单独点击「保存样例文字」。

### 缩略图重新生成

提供以下批量重生成按钮（按需点击）：

- 3D 模型缩略图
- 字体缩略图
- C4D / Max / Blend 嵌入预览图
- JSON / Markdown / TXT 文本预览缩略图

## 外观

| 设置项 | 说明 | 选项 |
|--------|------|------|
| 语言 | 界面显示语言，**立即生效** | 简体中文 / English |
| 主题 | 浅色或深色，**立即生效** | 浅色 / 深色 |
| 瀑布流网格大小 | 列宽预设：小（120px）/ 中（200px）/ 大（280px） | 中 |

网格大小保存后生效；在主界面还可通过 `Ctrl`（macOS 为 `⌘`）+ 滚轮微调列宽。

## 快捷键

快捷键标签页以只读列表展示常用快捷键。点击快捷键按钮目前**不支持自定义改键**（功能尚未实现）。

完整快捷键列表见 [快捷键速查](shortcuts.md)。设置页中仅显示 `showInSettings: true` 的子集。

## 高级

### Web API

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用 | 应用运行时是否监听 HTTP API | **开启** |
| 端口 | 监听端口 | 41596 |
| 允许远程访问 | 监听 `0.0.0.0` 并要求 Bearer Token | 关闭 |
| Token | 远程访问鉴权令牌，可重新生成 | 空 |

- 本机访问（未开启远程）：`127.0.0.1`，**无需 Token**
- 开启远程访问后必须设置 Token
- 可一键复制 Base URL、打开 API Playground

详见 [Web API 与浏览器扩展](web-api.md)。

### 资料库自检

对比 SQLite `assets` 表行数与 `items/` 子目录数量：

- 数量一致：显示正常
- 不一致：提示多余或缺失的 `items/` 目录

此检查**不会**扫描数据库完整性、缩略图缓存或 catalog 引用文件；引用问题请使用资料库标签页的「检查引用」。

### 日志级别

主进程日志详细程度：`error` / `warn` / `info` / `debug`。也可通过环境变量 `ASSETVAULT_LOG_LEVEL` 覆盖。

### 搜索延迟

输入搜索词后的防抖时间，范围 100–800 ms，默认 300 ms。

### 最大缓存

缩略图 LRU 内存缓存上限，范围 256–10240 MB，默认 2048 MB。
