# AI 画布

AI 画布是基于 **React Flow** 的节点式创作工作台，在**独立窗口**中运行（`#ai-canvas`），可与主资料库窗口并排使用。暂时不可使用

> v0.5.0-alpha：节点生成目前为**本地 Mock**（约 2 秒延迟 + 占位图），尚未接入生产级 AI 网关。规格见 [AI_GATEWAY_SPEC.md](../AI_GATEWAY_SPEC.md)。

---

## 打开方式

主窗口工具栏点击 **AI 画布** 按钮，会打开或聚焦独立窗口。

列表页可创建/删除画布，显示节点数与更新时间，并可切回资料库窗口。

---

## 画布能力

- 添加、连线、删除节点（`Delete` 删除选中节点；在编辑器中屏蔽资料库的删除快捷键）
- 从资料库窗口**拖放资产**到画布，自动创建对应素材节点
- 保存画布文档、返回列表

---

## 节点类型

注册于 `aiCanvasNodes.tsx`，添加菜单（`+` 按钮）当前暴露：

### 素材节点


| 类型         | 添加菜单 | 说明            |
| ---------- | ---- | ------------- |
| base_text  | ✅    | 文本素材          |
| base_image | ✅    | 图片素材          |
| base_video | ✅    | 视频素材          |
| base_audio | ❌    | 代码已注册，添加菜单未暴露 |


### 生成节点


| 类型                  | 说明   |
| ------------------- | ---- |
| generate_text       | 文本生成 |
| generate_image      | 图片生成 |
| generate_video      | 视频生成 |
| generate_audio      | 音频生成 |
| generate_storyboard | 分镜生成 |


另有旧版别名节点（`text`、`image`、`video`、`reference`、`output`）用于兼容已有画布。

连线规则由 `canConnectNodes` / `modeConfigCatalog.ts` 校验，详见 [node_type.md](../node_type.md)。

## 基本流程

### 1. 添加节点

画布左侧工具栏 `+` → 选择生成节点或素材节点。

### 2. 拖入资料库资产

在主窗口选中资产，拖到 AI 画布窗口，松开后创建图片/视频等素材节点。

### 3. 连线

从输出端口拖线到输入端口；不符合类型规则的连线会被拒绝。

### 4. 运行生成

点击生成节点上的运行按钮 → 当前实现调用 `runNodeGenMock`：

- 队列 → 运行中（进度条）→ 约 2 秒后完成
- 图片类输出为占位 PNG；**不消耗积分、不需要联网**

图片生成 Dock 可选择模型（如 Lib Nano Pro / Mock）、宽高比、批量数；风格/标注/镜头等标注为「即将推出」。

### 5. 输出

生成结果展示在节点上；后续版本将支持导入资料库（以实际 UI 为准）。

## 画布操作


| 操作   | 方式             |
| ---- | -------------- |
| 平移   | 中键拖动           |
| 缩放   | 滚轮             |
| 选择   | 左键             |
| 多选   | 框选 / `Ctrl`+点击 |
| 删除节点 | 选中后 `Delete`   |
| 连线   | 端口拖线           |
| 断线   | 选中连线后 `Delete` |
