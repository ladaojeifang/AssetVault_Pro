# 快捷键速查

快捷键定义以 `src/shared/hotkeyCatalog.ts` 为单一数据源；窗口聚焦时由渲染进程处理。可选的全局快捷键需设置环境变量 `ASSETVAULT_GLOBAL_HOTKEYS=1`（默认关闭，避免抢占其他应用按键）。

---

## 通用

| 快捷键 | 功能 |
|--------|------|
| `Space` | 仅当选中 **1** 个资产时，打开详情面板 |
| `Ctrl+K` | 聚焦搜索栏 |
| `Ctrl+,` | 打开设置 |
| `F5` | 刷新当前资产列表 |
| `Ctrl+R` | 刷新当前资产列表（与 F5 等效） |

## 导入

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+I` | 导入文件（打开文件选择器） |
| `Ctrl+Shift+O` | 导入文件夹（递归扫描支持的扩展名） |

## 界面

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 切换侧栏显示/隐藏 |
| `Ctrl+D` | 切换详情面板显示/隐藏 |
| `Ctrl+L` | 打开资料库切换器 |

## 视图

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+G` | 切换为网格视图 |
| `Ctrl+Shift+G` | 切换为列表视图 |
| `Ctrl+A` | 全选（焦点在输入框时不触发） |

## 编辑

| 快捷键 | 功能 |
|--------|------|
| `Delete` | 删除选中资产（需确认） |
| `Backspace` | 与 Delete 相同 |

在 AI 画布编辑器中，删除快捷键被屏蔽。

## 缩略图

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+T` | 自定义缩略图（从文件，需单选） |
| `Ctrl+Shift+Alt+T` | 自定义缩略图（从剪贴板，需单选） |
| `Ctrl+Alt+R` | 刷新选中资产的缩略图 |

缩略图快捷键也会出现在资产右键菜单中。

---

## 与设置页列表的差异

设置 → 快捷键标签页仅展示部分快捷键（`showInSettings: true`），不包含：`Ctrl+G`、`Ctrl+Shift+G`、`Ctrl+A`、`Space`、`Ctrl+,` 及缩略图相关快捷键。

设置页中的快捷键按钮目前**不支持改键**。
