# 快速入门

本指南帮助你安装 AssetVault Pro，创建第一个资料库，并导入、浏览资产。

---

## 1. 安装

### Windows

1. 从发布页面下载 `.exe` 安装包
2. 双击运行安装程序并完成向导
3. 启动 AssetVault Pro



## 2. 创建资料库

应用通过**资料库切换器**（侧栏顶部）管理资料库，没有单独的「欢迎向导」页面。

1. 点击侧栏顶部的资料库名称，打开资料库切换器
2. 选择「新建资料库」
3. 在对话框中选择模式：
   - **archive（完整库）**：在**空文件夹**中初始化；文件复制到资料库 `items/` 目录，保留资产副本、元数据及缩略图
   - **catalog（索引库）**：在**空文件夹**中初始化；仅记录外部文件路径，资料库 `items/` 目录只保留资产文件元数据及缩略图
   - **embedded（内嵌库）**：选择**已有项目文件夹**；文件保持原位，扫描支持的扩展名后入库，资料库 `items/` 目录只保留资产文件元数据及缩略图
4. 点击确认后，用系统文件夹选择器选定目标目录
5. 资料库显示名称默认取自文件夹名（或 manifest 中的 `displayName`）

> 新手推荐 **archive**，文件集中在资料库内，不怕误删原文件。

也可在 **设置 → 资料库** 标签页新建或切换资料库。

---

## 3. 导入资产

桌面端工具栏提供以下导入方式：

### 拖放

将文件或文件夹从资源管理器拖入主窗口。拖入文件夹时会递归导入其中所有受支持的扩展名。

若当前正在浏览某个文件夹，导入的资产会自动归入该文件夹。

### 工具栏 / 快捷键

| 操作 | 方式 |
|------|------|
| 导入文件 | 工具栏「导入文件」，或 `Ctrl+I` |
| 导入文件夹 | 工具栏「导入文件夹」，或 `Ctrl+Shift+O` |

### URL / 在线视频

桌面端**没有**「从 URL 导入」或「从作品页导入视频」的菜单项。这些能力通过 **Web API** 或 **浏览器扩展** 调用，详见 [Web API 与浏览器扩展](web-api.md)。

---

## 4. 浏览与搜索

- **网格 / 列表**：工具栏切换，或 `Ctrl+G` / `Ctrl+Shift+G`
- **搜索**：`Ctrl+K` 聚焦搜索栏；多词以空格分隔，AND 匹配
- **筛选**：工具栏下方筛选条（颜色、尺寸、文件大小、日期）；侧栏按文件类型、标签、文件夹筛选
- **排序**：工具栏排序下拉（导入时间、文件名、大小、类型、扩展名、浏览量、随机等）
- **详情**：选中单个资产后按 `Space`
- **全页预览**：双击资产（若格式支持预览）；否则在资源管理器中定位文件

---

## 5. 整理资产

### 标签

1. 选中资产，按 `Space` 打开详情面板
2. 在标签区域输入名称，回车创建或关联已有标签

侧栏可点击标签作为筛选条件；**没有**独立的「标签管理」界面。

### 文件夹

1. 侧栏文件夹树右键 →「新建子文件夹」
2. 将资产加入文件夹：
   - 详情面板文件夹下拉框
   - 资产右键菜单 →「添加到文件夹」
   - 拖放到侧栏文件夹时按住 **Alt**（主区域子文件夹卡片可直接拖放）
3. 浏览某文件夹时，可在资产右键菜单中「设为文件夹封面」
