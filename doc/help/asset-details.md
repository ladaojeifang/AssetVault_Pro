# 资产详情面板

详情面板用于查看和编辑单个资产的信息。

---

## 打开与关闭

| 方式 | 行为 |
|------|------|
| `Space` | 恰好选中 **1** 个资产时打开面板 |
| `Ctrl+D` | 切换详情面板显示/隐藏 |
| 面板右上角 × | 关闭面板 |

> `Space` 打开的是详情面板，**不是**全页预览。全页预览请双击资产，或在面板内点击对应预览按钮。

---

## 预览区域

顶部按资产类型展示预览：

| 类型 | 预览 |
|------|------|
| 图片 | 缩略图 |
| 3D（可预览格式） | 嵌入式 Babylon.js 交互预览 |
| 其他 | 格式占位图标 |

若资产支持全页预览，预览区下方会出现对应按钮（字体 / 3D / SVG / EXR / Markdown），点击进入主区域全页视图。

---

## 文件信息

| 字段 | 说明 |
|------|------|
| Asset ID | 资料库内唯一标识 |
| Type | `file_type`（image、video、3d 等） |
| Size | 文件体积 |
| Dimensions | 图片/视频宽高（如有） |
| Duration | 音频/视频时长（如有） |
| Imported | 导入时间 |
| Views | 浏览次数 |
| 存储方式 | 本地副本 / 外部引用 / 引用缺失 |
| 源路径 | catalog 引用模式下显示外部路径 |

`storageMode=embedded` 的资产在界面上显示为本地存储。

---

## 文件夹

资产可属于多个文件夹（多对多）。

- **查看**：显示已关联文件夹，点击 × 移除关联
- **添加**：通过「添加文件夹」下拉框选择目标文件夹

移除关联不会删除资产本身。

拖放归类：向侧栏文件夹拖放资产时需按住 **Alt**；主区域子文件夹卡片可直接拖放。

---

## 标签

- 显示当前标签，点击 × 解除关联
- 点击输入区，输入名称后回车：选中已有标签或**创建新标签并关联**

应用内**没有**独立的标签编辑/删除管理页；新建标签仅在详情面板中完成。侧栏标签用于筛选。

---

## 色彩分析

**图片与视频**资产会自动（或手动重新）分析主色调，展示：

- 10 色条调色板
- 主色色块与色值

可在资产右键菜单中选择「重新分析颜色」。

---

## 备注

- 点击备注区域编辑
- 失焦时自动保存
- 上限 **16000** 字符

---

## 来源链接

- 输入来源 URL，失焦保存
- 上限 **2048** 字符
- 以 `http://` 或 `https://` 开头时可点击「打开链接」

来源链接纳入搜索索引。

---

## 底部操作

| 操作 | 条件 | 说明 |
|------|------|------|
| 本地化 | `storageMode=referenced` | 将外部文件复制进资料库 |
| 重新链接 | 引用缺失时 | 选择新文件路径 |
| Open | 始终 | 用系统默认程序打开 |
| Show in Explorer | 始终 | 在资源管理器中定位 |
| Delete | 始终 | 从资料库删除（不可恢复） |

「本地化」仅在 catalog 引用模式下显示。「重新链接」仅在检测到源文件缺失时出现。
