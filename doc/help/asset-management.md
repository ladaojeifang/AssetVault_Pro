# 资产管理

了解如何在 AssetVault Pro 中导入、浏览、搜索和筛选数字资产。

---

## 支持的格式

当前可导入扩展名以 `src/shared/assetFormatCatalog.ts` 为准，合计 **150+** 种。`file_type` 分为：

`image` · `video` · `audio` · `font` · `design` · `document` · `3d` · `code` · `other`

完整清单见 [asset-types-and-import.md](../asset-types-and-import.md)。

---

## 导入资产

### 桌面端导入

| 方式 | 说明 |
|------|------|
| 拖放 | 拖入文件或文件夹到主窗口 |
| `Ctrl+I` | 多选文件导入 |
| `Ctrl+Shift+O` | 递归扫描文件夹内受支持扩展名 |

导入到当前浏览的文件夹时，新资产会自动关联该文件夹。

文件夹导入仅处理 `ALL_SUPPORTED_IMPORT_EXTENSIONS` 中的扩展名；子目录会一并扫描。

### Web API / 浏览器扩展

以下导入方式**不在**桌面端菜单中，需通过 API 或 [浏览器扩展](../browser-extension.md)：

| 能力 | API 端点（示例） |
|------|------------------|
| 单 URL 导入 | `POST /api/v1/asset/importFromURL` |
| 批量 URL | `POST /api/v1/asset/importFromURLBatch` |
| Data URL（截图等） | `POST /api/v1/asset/importFromDataUrl` |
| 整页长截图会话 | `POST /api/v1/asset/fullPageSession/*` |
| Markdown 资料包会话 | `POST /api/v1/asset/articleBundleSession/*` |
| 作品页视频（yt-dlp） | `POST /api/v1/asset/pageVideoImport` |

作品页 URL 不可通过普通 `importFromURL` 导入，须走 `pageVideoImport`。

### 存储模式与导入行为

| 资料库模式 | 导入后文件位置 |
|------------|----------------|
| archive | 复制或硬链接到 `items/{id}/` |
| catalog | 保留外部路径引用（`storageMode=referenced`） |
| embedded | 文件保持原位（`storageMode=embedded`） |

OBJ 导入时，同行的 MTL 会一并复制，但不会作为独立资产入库。

## 导入去重

检测方式：

- **相同导入源路径**：相同 canonical 路径再次导入时跳过（除非策略为再导入一份）
- **SHA-256 内容哈希 + 文件大小**：识别内容相同的文件

### 桌面端（默认 `ask`）

弹出对话框，可选择：

- **使用已有**：关联到已有资产（可勾选「应用于本批其余文件」）
- **再导入一份**：在资料库中新建副本
- **取消**：跳过当前文件

**设置中无法配置默认重复策略**；每次导入批次可在对话框中勾选「应用于全部」。

### Web API（默认 `use_existing`）

未指定 `duplicatePolicy` 时，内容重复默认使用已有资产，不弹窗。可传 `import_copy` 强制新建副本。

可在 **设置 → 资料库 → 内容指纹扫描** 中补全历史资产的哈希，以改善去重效果。

## 浏览资产

### 瀑布流网格

- 虚拟滚动，大量资产下仍较流畅
- 列宽：设置中预设小/中/大，或 `Ctrl`（macOS `⌘`）+ 滚轮微调
- 无缩略图时显示格式占位图标（可在设置中自定义）

### 列表视图

表头支持按列排序；「类型」列下拉可筛选系统格式或用户自定义分类（与侧栏共用筛选状态）。

### 默认打开行为

- **双击**：若支持全页预览则进入预览；否则在资源管理器中定位
- **Space**：打开详情面板（需恰好选中 1 个资产）

### 排序

工具栏下拉支持：

| 字段 | 说明 |
|------|------|
| importedAt | 导入时间（默认：最新优先） |
| filename | 文件名 |
| fileSize | 文件大小 |
| fileType | 文件类型 |
| extension | 扩展名 |
| viewCount | 浏览次数 |
| random | 随机 |

## 搜索

按 `Ctrl+K` 聚焦搜索栏。

搜索索引字段包括：**文件名**、**原始文件名**、**标签名**、**备注**、**来源链接**（`assets_search.search_text`）。

多个关键词以空格分隔，采用 **AND** 逻辑（须同时匹配所有词）。

搜索输入有防抖延迟，默认 300 ms，可在设置 → 高级中调整。

## 筛选

### 侧栏

- **类型**：同一列表包含 9 个系统格式（按扩展名自动判定，不可重命名）与用户自定义分类（可右键编辑/删除）；可多选，结果为 **OR**（匹配任一所选类型即显示）
- **标签**：多选标签，结果为并集（包含任一所选标签即显示）
- **文件夹**：进入文件夹即限定范围

列表视图表头「类型」列为单选下拉，选项与侧栏一致；侧栏多选时列头显示「全部」。**扩展名**列支持下拉选择资料库已有扩展名，或手动输入（如 `png` / `.png`）。

### 筛选条（网格视图：工具栏下方；列表视图：表头第二行）

| 筛选项 | 说明 |
|--------|------|
| 主色调 | 按色彩桶（color bucket）匹配 |
| 尺寸预设 | 图片宽高的预设档位 |
| 文件大小 | 预设或自定义 MB 范围 |
| 扩展名 | 精确匹配扩展名；下拉或输入 |
| 导入日期 | 今天 / 本周 / 本月 / 今年等 |

## 选择与批量操作

- `Ctrl`（macOS `⌘`）+ 点击：多选
- `Shift` + 点击：范围选择
- `Ctrl+A`：全选

选中多个资产后，可通过**右键菜单**执行：

- 在资源管理器中打开
- 添加到文件夹 / **复制到其他资料库**（含文件夹、标签、有效类型 `type_id`；无内容去重）
- 设为当前文件夹封面（浏览文件夹时，且资产为图片或有缩略图）
- 重命名（仅单选）
- 复制文件 / 路径到剪贴板
- 重新分析颜色（图片或视频）
- 自定义缩略图、刷新缩略图
- 删除（需确认）

类型、标签、备注等编辑主要在详情面板中进行。整库合并见设置 → 资料库 → **从其他资料库导入**（自动合并标签与 `type_id`）。
